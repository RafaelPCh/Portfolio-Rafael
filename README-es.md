## 🚀 Proyectos

### [Predicción de Abandono de Clientes (Customer Churn)](./Churn-Prediction)
* **Objetivo:** Predecir qué clientes tienen un alto riesgo de abandonar el servicio utilizando machine learning.
* **Tecnologías:** Python, scikit-learn, pandas, joblib.
* **Descripción:** Un pipeline completo de machine learning que entrena un modelo Random Forest con datos de clientes para identificar indicadores de abandono.
